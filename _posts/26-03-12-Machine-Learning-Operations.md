This Project is about building an end-to-end machine learning workflow to predict housing prices  using the California Housing dataset.
The focus is on applying key concepts: preprocessing, cross-validation, hyperparameter tuning, pipeline construction, and model deployment
The task involved loading the dataset:
Applying preprocessing techniques.
Splitting the data into training and test sets (80/20 split).
Building the model and Evaluating the results.
Saving the trained pipeline using pickle or joblib.
